1. 基于vue的markdown编辑器
   1. 安装 npm i mavon-editor --save
   2. 在main.js中引入
      1. import mavonEditor from 'mavon-editor'
      2. import "mavon-editor/dist/css/index.css"
      3. Vue.use(mavonEditor)
   3. 编辑器设置
      ```
      <mavon-editor v-model="content"
                    defaultOpen="preview"
                    :toolbarsFlag="false"
                    :subfield="false"></mavon-editor>
      ```
2. 页面刷新
   1. location.reload()
